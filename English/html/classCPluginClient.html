Detailed Description

The plugin interface.

Create CConnecter instance.

  • blocking model: The interface only is implemented by plug-in. It may be used with CConnecterDesktopThread for connection is blocking model.
  • no-blocking model: CPluginClientThread with CConnecterDesktop for a single background thread handles multiple remote desktop connections. (Not used yet)
