Detailed Description

Connecter interface.

Note
  • The class is a interface used by Use UI. It is implemented by the Protocol plugin.
  • Its instance is in the main thread.

Sequen diagram:

Basic implementations of the following types have been provided:

  1. Desktop type:
    1.1. The connection used is the blocking model (One thread handles one connection): CConnecterDesktopThread
    1.2. The connection is a non-blocking model (One thread handles multiple connections): CConnecterDesktop
  2. Termianal type: CConnecterTerminal
