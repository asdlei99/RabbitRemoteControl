Detailed Description

A widget which displays output image from a CConnect and sends input keypresses and mouse activity to the CConnect.

  • When the viewer receives new image output from the CConnect, it will update the display by calling slotUpdateRect.
  • When the mouse update, then call slotUpdateCursor ; if update position, then call slotUpdateCursorPosition
